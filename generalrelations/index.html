Ontology Alignment Evaluation Initiative - OAEI-2025 Campaign

GeneralRelation

General description

The GeneralRelation track aims to evaluate ontology matching systems on their ability to identify a wider range of semantic relations beyond equivalence. The benchmark includes subclass, superclass, overlap, and disjoint relations to reflect real-world ontology alignment scenarios.

This track addresses the growing demand for semantic integration where simple equivalence mappings are insufficient. It provides enriched datasets derived from product classification standards, STROMA/TaSeR test cases, and modified OAEI tracks, especially Anatomy.

Datasets

Participants can download the complete benchmark dataset via Zenodo:

  • https://zenodo.org/records/15682752

Three major sources are used:

  • Product Classification Standards (e.g., ETIM, eClass, UNSPSC, GPC)
  • STROMA/TaSeR manually annotated test cases with various relation types
  • Modified OAEI Testcases with injected subclass/superclass relations using enrichment strategies

Evaluation follows the isAmong framework alignment framework, which measures semantic agreement between predicted and reference alignments at the leaf-class level, rewarding partially correct correspondences.
