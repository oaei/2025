Ontology Alignment Evaluation Initiative - OAEI-2025 Campaign

CE

General description

The CE track consists of finding alignments between relevant Circular Economy ontologies.

The CE track in 2025 consists of 2 task. The frist task is to match 2 ontologies in the Circular Economy domain, with the standard configuration of your matching system. The two ontologies are the Circular Economy Ontology Network (CEON) [3] and Sustainable Bioeconomy and Bioproducts Ontology (BiOnto) [4]. The second task is to the CE-related ontology, CEON, with a materials domain ontology, MatOnto [5].

The general schedule given at the OAEI main page is binding for the anatomy track. Moreover, you have to submit your wrapped tool as explained on the OAEI main page.

Data sets

It is highly recommended that you run local tests to see whether your matching system can cope with the dataset. There are two options:

(1) You can download the whole dataset and run your tests with this input data.

(2) However, finally you have to create a package or a Web service of your tool based on MELT. For this purpose, a tutorial is linked via the OAEI 2025 page (MELT evaluation instructions). For running the CE testsuites you will have to specify the following input parameters.

CE Testsuite
  • Repository: http://oaei.webdatacommons.org/tdrs/
  • Suite-ID: ce_track
  • Version-ID: v2
