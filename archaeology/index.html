Ontology Alignment Evaluation Initiative - OAEI-2025 Campaign

Multilingual Archaeology

General description

The multilingual archaeology track is participating for the second time at the OAEI. It aims at the alignment of multilingual domain specific vocabularies. The high quality reference is compiled manually, ensuring semantic, lexical and part-of-speech similarity. It offers different test cases from the archaeology domain. Each source and target ontology is monolingual in one of the following languages: English, German, French, Italian. The goal is to evaluate the matching system perfomance when dealing with small datasets in different languages and specialist terms.

Dataset description

The resources and a comprehensive description is available in the Multilingual archaeology benchmark github repo. Each test case consists of a source and a target ontology and a manual reference alignment. For data provenance, see github description for more details.

Archaeology

There are ten different test cases available. All of them use DEFC as source and PACTOLS as target.

  • de-de
  • de-en
  • de-fr
  • de-it
  • en-en
  • en-fr
  • en-it
  • fr-fr
  • fr-it
  • it-it

Modalities

MELT will be used and the output of the correspondences will automatically be evaluated against the manually complied reference by MELT. The evaluation will be based on precision, recall and F-measure. There will be additional criteria such as max. memory consumption and runtime.
